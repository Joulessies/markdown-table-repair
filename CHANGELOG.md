# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-16

### Added
- Initial release
- `repairTable(input)` function to fix broken Markdown tables
- Support for CommonJS, ES Modules, and UMD (browser)
- TypeScript type definitions
- Handles missing pipes, mismatched columns, and incomplete separator rows
- Works with streaming/partial tables from AI responses
